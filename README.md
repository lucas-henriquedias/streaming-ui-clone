# 🎬 Streaming UI Clone

Projeto de estudo inspirado na interface da Netflix, com foco na tela de seleção de perfis e navegação para catálogo de filmes.

---

## 📌 Sobre o projeto

Este projeto foi iniciado a partir de uma Imersão de Desenvolvimento Front-End oferecida pela plataforma Alura.

A tela de seleção de perfis foi desenvolvida por mim, incluindo estrutura HTML, estilização em CSS, organização e responsividade de layout.

A lógica em JavaScript (alternância de tema claro/escuro) e a estrutura inicial da tela de catálogo foram fornecidas durante a imersão e posteriormente integradas, adaptadas e organizadas por mim no projeto.

Projeto sem fins comerciais, desenvolvido exclusivamente para fins de aprendizado e prática de front-end.

---

## 🚀 Funcionalidades

- Seleção de perfis de usuário
- Navegação para tela de catálogo
- Alternância entre tema claro e escuro
- Layout responsivo para diferentes tamanhos de tela

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox e responsividade)
- JavaScript
- Visual Studio Code

---

## 🎯 Aprendizados

Durante o desenvolvimento, pratiquei e aprimorei:

- Estruturação semântica com HTML
- Organização e reutilização de CSS
- Responsividade com media queries
- Leitura e adaptação de código já existente
